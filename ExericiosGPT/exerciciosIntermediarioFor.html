1) Tabuada completa com arrays Peça ao usuário um número e armazene no array os resultados da tabuada desse número (de 1 a 10). Exemplo: se digitar 5, o array deve conter [5, 10, 15, ..., 50].

2) Inverter um array Crie um array com números de 1 a 10. Use um for para gerar um novo array invertido (de 10 a 1).

3) Somar apenas os números pares Peça ao usuário 10 números, armazene-os em um array e, ao final, calcule a soma apenas dos pares.
